## 版本说明

### 版本号格式

版本号格式：`主版本号.次版本号.修订号`

- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型

- `新增` (Added): 新功能
- `修改` (Changed): 现有功能的变更
- `弃用` (Deprecated): 即将移除的功能
- `移除` (Removed): 已移除的功能
- `修复` (Fixed): Bug 修复
- `安全` (Security): 安全相关的修复

### 图标说明

- ✨ 新功能
- 🐛 Bug 修复
- 🔧 配置修改
- 📝 文档更新
- 🎨 样式调整
- ⚡ 性能优化
- 🔒 安全修复
- 🎯 功能增强
- 📚 文档完善
- 🚀 部署相关
